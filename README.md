# httpc
### A C library for HTTP servers

#### Purpose
This project was mostly made as an exercise to get comfortable with lower level languages rather than out of an actual need to create an quality, performant, ready-to-use HTTP library. While this should be obvious, httpc has absolutely no use in a production environment, so please don't use it as such! :)
