# httpc
### A "minimal" C library for HTTP servers
#### (p.s: don't actually use this)

#### Purpose
C, being such a popular language, already has a multitude of uses in web technology, including individual servers, clients, and frameworks/libraries with those. However, these libraries often require tons of boilerplate and give lots of things not really needed or desired. httpc serves as a minimal library for this purpose, providing (hopefully) worthwhile performance along with ease of use with C.

This project was mostly made as an exercise to get comfortable with lower level languages rather than out of an actual need to create an quality, performant, ready-to-use HTTP library. While this should be obvious, httpc has absolutely no use in a production environment, so please don't use it as such! :)
